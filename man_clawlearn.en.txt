clawlearn(1) -- Build Anki cloze decks from local text/EPUB files

SYNOPSIS
    python -m clawlearn.cli [GROUP] COMMAND [OPTIONS]

DESCRIPTION
    ClawLearn converts real-world text (podcast transcripts, articles,
    textbook examples) into Anki cloze decks (.apkg).

    Input is a local .txt/.md/.epub file. Web/URL fetching should be
    handled upstream (e.g. by clawfetch), producing Markdown or plain
    text for ClawLearn.

COMMANDS
    init
        Initialize runtime files.

        - create .env from .env.example if missing
        - verify prompts/templates exist:
          ./prompts/cloze_contextual.json
          ./prompts/cloze_textbook_examples.json
          ./prompts/translate_rewrite.json
          ./templates/anki_cloze_default.json

    doctor
        Check dependencies, config, prompts, LLM connectivity, TTS, and
        output directory.

        Usage:
            python -m clawlearn.cli doctor [--env-file FILE]

    lingua build deck
        Build an Anki .apkg from a local file (lingua pipelines).

        Usage:
            python -m clawlearn.cli lingua build deck INPUT \
              --source-lang CODE \
              --target-lang CODE \
              [--material-profile prose_article|transcript_dialogue|textbook_examples] \
              [--learning-mode lingua_expression|lingua_reading] \
              [--content-profile <deprecated>] \
              [--input-char-limit N] \
              [--env-file FILE] \
              [--output PATH] \
              [--deck-name NAME] \
              [--max-chars N] \
              [--cloze-min-chars N] \
              [--max-notes N] \
              [--temperature FLOAT] \
              [--difficulty beginner|intermediate|advanced] \
              [--prompt-lang en|zh] \
              [--extract-prompt PATH] \
              [--explain-prompt PATH] \
              [--lingua-annotate/--no-lingua-annotate] \
              [--lingua-annotate-batch-size N] \
              [--lingua-annotate-max-items N] \
              [--save-intermediate/--no-save-intermediate] \
              [--continue-on-error] \
              [--verbose] \
              [--debug]

        INPUT
            Path to .txt/.md/.epub file.

        OPTIONS
            --source-lang
                Override source language (default from env).

            --target-lang
                Override target language (default from env).

            --material-profile
                Material profile. Supported values:
                    prose_article | transcript_dialogue | textbook_examples

            --learning-mode
                Learning mode selector.
                - lingua: lingua_expression | lingua_reading
                - textbook: textbook_focus | textbook_review

            --content-profile
                (Deprecated) Alias of --material-profile. Legacy `general` maps to `prose_article`.

            --input-char-limit
                Only process the first N characters of input. Useful for
                quick test runs (e.g. --input-char-limit 4000).

            --env-file
                Path to .env file. If omitted, default resolution logic is used.

            --output
                Output .apkg path. Defaults to outputs/<run_id>/output.apkg.

            --deck-name
                Override Anki deck name. Defaults to CLAWLEARN_DEFAULT_DECK_NAME
                or the input file stem.

            --max-chars
                Override CLAWLEARN_CHUNK_MAX_CHARS for this run.

            --cloze-min-chars
                Override CLAWLEARN_CLOZE_MIN_CHARS for this run.

            --max-notes
                Limit total number of exported notes.

            --temperature
                LLM temperature (overrides env).

            --difficulty
                Override CLAWLEARN_CLOZE_DIFFICULTY. Supported values:
                    beginner | intermediate | advanced.

            --prompt-lang
                Prompt language for multi-lingual prompts (en|zh).

            --save-intermediate / --no-save-intermediate
                Enable/disable dumping intermediate JSONL files under
                CLAWLEARN_OUTPUT_DIR/<run_id>.

            --continue-on-error
                Continue on individual cloze/translation/TTS errors and
                record them in errors.jsonl instead of aborting.

            --debug
                Re-raise internal exceptions with full traceback.

    prompt validate
        Validate a prompt JSON file against the expected schema.

        Usage:
            python -m clawlearn.cli prompt validate PATH

    config show
        Show merged runtime configuration as JSON.

        Usage:
            python -m clawlearn.cli config show [--env-file FILE]

    config validate
        Validate configuration without building a deck.

        Usage:
            python -m clawlearn.cli config validate [--env-file FILE]

ENVIRONMENT
    CLAWLEARN_LLM_*                     OpenAI-compatible LLM configuration
    CLAWLEARN_TRANSLATE_LLM_*           Optional small LLM for translation
    CLAWLEARN_SECONDARY_EXTRACT_*       Optional secondary extraction pass (dual-LLM)
    CLAWLEARN_CHUNK_*                   Chunking controls
    CLAWLEARN_CLOZE_*                   Cloze-level controls
    CLAWLEARN_PROMPT_LANG               Default prompt language (en|zh)
    CLAWLEARN_MATERIAL_PROFILE          Default material profile
    CLAWLEARN_LEARNING_MODE             Default learning mode
    CLAWLEARN_CONTENT_PROFILE           Legacy alias (maps to material profile)
    CLAWLEARN_TTS_*                     TTS provider and voice lists
    CLAWLEARN_OUTPUT_DIR                Output directory for run data (JSONL, media)
    CLAWLEARN_EXPORT_DIR                Output directory for exported decks
    CLAWLEARN_LOG_DIR                   Directory for log files
