clawlearn(1) -- 从本地文本/EPUB 构建 Anki 填空牌组

用法
    python -m clawlearn.cli [分组] 命令 [选项]

说明
    ClawLearn 将真实语料（播客字幕、文章、教材例句）转换为
    Anki 填空牌组（.apkg）。

    输入为本地 .txt/.md/.epub 文件。网页抓取请交给上游工具
    （例如 clawfetch），生成 Markdown 或纯文本后再交给 ClawLearn。

命令
    init
        初始化运行环境。

        - 若不存在 .env，则从 .env.example 复制一份；
        - 校验以下文件是否存在：
          ./prompts/cloze_contextual.json
          ./prompts/cloze_textbook_examples.json
          ./prompts/translate_rewrite.json
          ./templates/anki_cloze_default.json

    doctor
        检查依赖、配置、prompt、LLM 连通性、TTS 和输出目录。

        用法：
            python -m clawlearn.cli doctor [--env-file FILE]

    lingua build deck
        从本地文件构建 Anki .apkg 牌组（lingua pipelines）。

        用法：
            python -m clawlearn.cli lingua build deck INPUT \
              --source-lang CODE \
              --target-lang CODE \
              [--material-profile prose_article|transcript_dialogue|textbook_examples] \
              [--learning-mode lingua_expression|lingua_reading] \
              [--content-profile <deprecated>] \
              [--input-char-limit N] \
              [--env-file FILE] \
              [--output PATH] \
              [--deck-name NAME] \
              [--max-chars N] \
              [--cloze-min-chars N] \
              [--max-notes N] \
              [--temperature FLOAT] \
              [--difficulty beginner|intermediate|advanced] \
              [--prompt-lang en|zh] \
              [--extract-prompt PATH] \
              [--explain-prompt PATH] \
              [--lingua-annotate/--no-lingua-annotate] \
              [--lingua-annotate-batch-size N] \
              [--lingua-annotate-max-items N] \
              [--save-intermediate/--no-save-intermediate] \
              [--continue-on-error] \
              [--verbose] \
              [--debug]

    textbook build deck
        从本地文件构建 Anki .apkg 牌组（textbook pipelines）。

        用法：
            python -m clawlearn.cli textbook build deck INPUT \
              --source-lang CODE \
              --target-lang CODE \
              [--learning-mode textbook_focus|textbook_review] \
              [--input-char-limit N] \
              [--env-file FILE] \
              [--output PATH] \
              [--deck-name NAME] \
              [--max-chars N] \
              [--max-notes N] \
              [--save-intermediate/--no-save-intermediate] \
              [--continue-on-error] \
              [--verbose] \
              [--debug]

    build deck
        （已弃用）旧的别名命令，建议使用 `lingua build deck`。

        用法：
            python -m clawlearn.cli build deck INPUT \
              --source-lang CODE \
              --target-lang CODE \
              [--content-profile <deprecated>] \
              [--material-profile prose_article|transcript_dialogue|textbook_examples] \
              [--learning-mode lingua_expression|lingua_reading] \
              [--input-char-limit N] \
              [--env-file FILE] \
              [--output PATH] \
              [--deck-name NAME] \
              [--max-chars N] \
              [--cloze-min-chars N] \
              [--max-notes N] \
              [--temperature FLOAT] \
              [--difficulty beginner|intermediate|advanced] \
              [--prompt-lang en|zh] \
              [--save-intermediate/--no-save-intermediate] \
              [--continue-on-error] \
              [--verbose] \
              [--debug]

        INPUT
            本地文件路径，支持 .txt/.md/.epub。

        选项
            --source-lang
                覆盖 env 中的源语言（默认从配置读取）。

            --target-lang
                覆盖 env 中的目标语言。

            --material-profile
                语料策略，支持：
                    prose_article | transcript_dialogue | textbook_examples

            --learning-mode
                学习模式选择：
                - lingua：lingua_expression | lingua_reading
                - textbook：textbook_focus | textbook_review

            --content-profile
                （Deprecated）等价于 --material-profile；旧值 `general` 会映射为 `prose_article`。

            --input-char-limit
                仅处理输入前 N 个字符，便于快速测试（例如
                --input-char-limit 4000）。

            --env-file
                指定 .env 文件路径；省略时使用默认查找逻辑。

            --output
                输出 .apkg 路径。默认 outputs/<run_id>/output.apkg。

            --deck-name
                覆盖默认的牌组名称。默认值为
                CLAWLEARN_DEFAULT_DECK_NAME 或输入文件名（去掉扩展名）。

            --max-chars
                覆盖本次运行的 CLAWLEARN_CHUNK_MAX_CHARS。

            --cloze-min-chars
                覆盖本次运行的 CLAWLEARN_CLOZE_MIN_CHARS。

            --max-notes
                限制导出的总卡片数。

            --temperature
                LLM 温度（覆盖 env）。

            --difficulty
                覆盖 CLAWLEARN_CLOZE_DIFFICULTY，取值：
                    beginner | intermediate | advanced

            --prompt-lang
                覆盖 CLAWLEARN_PROMPT_LANG，用于选择多语言 prompt 文案。
                典型取值：en | zh。

            --save-intermediate / --no-save-intermediate
                是否在 CLAWLEARN_OUTPUT_DIR/<run_id> 下输出
                中间 JSONL 文件（chunks、候选、翻译、最终卡片等）。

            --continue-on-error
                遇到单条 cloze/翻译/TTS 错误时继续执行，并将错误
                记录到 errors.jsonl，而不是整批失败。

            --debug
                出错时抛出完整 traceback，便于调试。

    prompt validate
        校验 prompt JSON 文件是否符合预期 schema。

        用法：
            python -m clawlearn.cli prompt validate PATH

    config show
        以 JSON 形式展示合并后的运行配置。

        用法：
            python -m clawlearn.cli config show [--env-file FILE]

    config validate
        校验配置是否有效但不构建牌组。

        用法：
            python -m clawlearn.cli config validate [--env-file FILE]

环境变量
    CLAWLEARN_LLM_*             主 LLM 配置（OpenAI-compatible）
    CLAWLEARN_TRANSLATE_LLM_*   翻译小模型配置（可选）
    CLAWLEARN_CHUNK_*           切块参数
    CLAWLEARN_CLOZE_*           cloze 粒度控制
    CLAWLEARN_PROMPT_LANG       默认 prompt 语言（en|zh）
    CLAWLEARN_MATERIAL_PROFILE  默认语料策略
    CLAWLEARN_LEARNING_MODE     默认学习模式
    CLAWLEARN_CONTENT_PROFILE   旧别名（会映射到 material_profile）
    CLAWLEARN_TTS_*             TTS 提供方与 voice 列表
    CLAWLEARN_OUTPUT_DIR        输出目录（中间运行数据 JSONL/媒体）
    CLAWLEARN_EXPORT_DIR        导出牌组目录
    CLAWLEARN_LOG_DIR           日志目录
